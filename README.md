# ZenPulse
A decentralized meditation platform that tracks user sessions and rewards mindful activities with ZEN tokens.

## Features
- Track meditation sessions
- Earn ZEN tokens for completing sessions
- User mood tracking and analytics
- Session history and statistics
- Customizable meditation profiles

## Contracts
The project consists of two main contracts:
- ZenToken: Implements the reward token system
- ZenPulse: Handles meditation sessions and user profiles

## Testing
Run tests using Clarinet:
```bash
clarinet test
```
